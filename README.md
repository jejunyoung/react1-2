#### 유의적 버전 
- 개발자가 사용하는 버전
- 호환을 위해 사용하는 버전 생성 방법

#### 사용법 (Ex: 3.14.5)
- 첫번째 자리는 배포한 번호(1.0.0)
- 두번째 자리는 기능을 추가한 번호(0.1.0)
- 마지막 자리는 오류를 수정한 번호(0.0.1)
- 앞 번호가 올라가면 뒤에 번호는 초기화
